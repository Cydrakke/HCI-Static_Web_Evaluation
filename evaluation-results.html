Usability Issues

The banner occupied half of the website's display.

Heuristic Violated: Consistency and Standards

Description:

Banner's size is too big, resulting users may focus more on banner instead of the items.

Impact on User Experience:

User might get annoyed by the size of banner, and might eventually quit website because the banner get more attention than the actual products.

Color theme between side menu (purple) and header (pink) is inconsistent.

Heuristic Violated: Consistency and Standards

Description:

Color inconsistent: While both color are vibrant, they belong to different color families and both don't blend smoothly. Which breaks the visual harmony expected in a cohesive design system.

Impact on User Experience:

Confusion and Distraction, visual inconsistency can confuse users by making the layout feel disjointed. Instead of focusing on their tasks, user may be distracted by the clashing design.

Does not provide confirmation before saving the entry

Heuristic Violated: Error prevention

Description:

when i was uodating an entry tried to save it it does not provide a confirmation before doing the action.

Impact on User Experience:

Users might feel frustrated because maybe the update contains an important part of the system that they dont want to update. this also gives less control to the user and make them less important

Lack of visual cues just a plain text interface

Heuristic Violated: Visibility of System Status, Recognition Rather Than Recall, Aesthetic and Minimalist Design

Description:

The site uses only plain text with no icons, buttons, or layout structure to guide users.

Impact on User Experience:

Users may feel confused or overwhelmed due to the lack of clear navigation and interaction hints. This can lead to frustration, increased errors, and early abandonment of the site, especially for first-time or non-technical users.

Absence of On-Screen return control from categories to sub categories

Heuristic Violated: User Control and Freedom, Visibility of System Status, Consistency and Standards

Description:

When I clicked or open the Accessories, I noticed that there is no visible button or link to return from a category page to its subcategories.

Impact on User Experience:

Users may feel trapped or forced to use the browser's back button. This disrupts navigation flow, increases frustration, and reduces efficiency—especially for users exploring multiple sections.

No On-Screen Help or Hints

Heuristic Violated: Help and Documentation, Error Prevention, Recognition Rather Than Recall

Description:

When I am browsing or exploring the List Products by Categories or the All products category the site provides no tooltips, help buttons, or guidance for users while navigating or using features.

Impact on User Experience:

Users may struggle to understand how to use certain features or complete tasks, leading to confusion, mistakes, and a poor overall user experience—especially for new users.

Lack of currency symbols.

Heuristic Violated: Visibility of System Status, Match Between System and the Real World

Description:

The site does not specify what currency symbol it uses when showing the price of an item.

Impact on User Experience:

This causes hesitation and distrust — users may abandon the cart rather than risk paying an unexpected amount. This small oversight raises redflags about how tru stworthy or secure the payment process will be.

There is no "Undo" button right after clicking the "Empty Cart" or "+" and "-" which can be frustrating in some cases.

Heuristic Violated: User Control and Freedom, Visibility of System Status, Help Users Recognize, Diagnose, and Recover from Errors

Description:

When clicking some buttons there are no undo which may cause frustration if it’s accidentally clicked.

Impact on User Experience:

Users feel powerless and may lose trust in the system. Frustration rises, especially if the cart had many items or specific quantities. Increased cart abandonment or support inquiries.

The system allows checkout even with zero items to checkout.

Heuristic Violated: Visibility of System Status, Error Prevention, Help Users Recognize, Diagnose, and Recover from Errors

Description:

Even when there are no items in the Cart section the system still allows checking out.

Impact on User Experience:

User confusion and frustration. Increased abandonment at checkout. Perceived sloppiness or brokenness of the app
